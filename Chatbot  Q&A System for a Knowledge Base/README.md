
## What technologies are used for this project?

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

web app: a chatbot / Q&A interface for a knowledge base.  
Features:  
- Admin can upload documents / PDFs / articles (knowledge base).  
- The system ingests and indexes content for semantic search / retrieval.  
- Users can ask natural language questions; bot returns best matching answers (with citation or link to original document).  
- Chat history, ability to refine or follow up.  
- Basic UI: a side panel showing related articles/links.  
- Responsive design (desktop + mobile).  
- Authentication: optional (users sign in to view premium content).  
- Admin dashboard: manage documents, view queries, analytics (which questions are frequent).  
- Dark / light mode toggle.
